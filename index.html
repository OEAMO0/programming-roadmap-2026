خريطة عربية تفاعلية مركزة على لينكس والأنظمة

خريطة تعلم البرمجة 2026

صفحة عربية مرتبة تقودك من 0 و1 والمنطق والتمثيل الثنائي إلى لينكس اليومي، ثم برمجة الأنظمة، النواة، وبناء التوزيعات، مع خريطة تفاعلية واضحة ومراجع أولية موثوقة.

الواجهة الآن تبدأ بصفحة هبوط حقيقية للقراءة والاختيار، ثم تنتقل إلى الخريطة التفاعلية في /map.

أبرز المسارات داخل الخريطة

الأساسيات العميقة من البتات والبوابات والتمثيل الثنائي إلى الذاكرة والتنفيذ والترجمة. Linux Userland / Operations الملفات، الصلاحيات، الشِل، النصوص، الخدمات، السجلات، والإقلاع المبكر. Linux Systems Programming C وlibc وELF والخيوط والمقابس والتتبع وأدوات التشخيص داخل لينكس. Linux Kernel Internals بناء النواة، UAPI، subsystems، التعريفات، debugging، ومسار الإقلاع. Linux Distribution Engineering Linux From Scratch وBuildroot وYocto والحزم والصور والتحديث والصيانة. Frontend / Web مسار ويب واحد أخف لكنه واضح، يشمل HTML وJavaScript والتخزين والاختبارات.

ما الذي يقدمه الموقع للمتعلم

  • صفحة دخول مفهومة بدل فلاش سريع يختفي قبل أن تقرأه.
  • خريطة تفاعلية مستقلة في /map مع فلترة صغيرة وفهرس جانبي ومفضلة وآخر ما فُتح.
  • تركيز أكبر بكثير على لينكس: الاستخدام اليومي، برمجة الأنظمة، النواة، وبناء التوزيعات.
  • إزالة المسارات الضعيفة من الواجهة الظاهرة لصالح عمق أعلى ومصادر أولية أوضح.
